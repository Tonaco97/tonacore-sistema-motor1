# Arquitetura do TonaCore Cyber BR

A arquitetura é baseada em camadas independentes e extensíveis:

- Core Fusion Engine
- Correlation Layer
- Policy Engine
- Integration Adapters
- Intelligence Layer
- Compliance Layer
- APIs (REST, GraphQL, Streaming)

Cada camada pode evoluir de forma independente, garantindo escalabilidade
e isolamento de responsabilidades.
